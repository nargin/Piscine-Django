Ex01 : Django, framework web

Qu'est-ce que Django ?

Django est un framework web open source écrit en Python qui suit le modèle architectural MVC (Modèle-Vue-Contrôleur), ou plus précisément le modèle MVT (Modèle-Vue-Template). Il a été conçu pour permettre un développement web rapide, sécurisé et maintenable.

Caractéristiques principales

  • Rapidité de développement : Django fournit de nombreuses fonctionnalités prêtes à l'emploi, ce qui permet aux développeurs de créer des applications rapidement.
  • Sécurité : Django inclut de nombreuses protections contre les vulnérabilités courantes du web comme les injections SQL, le cross-site scripting (XSS), le cross-site request forgery (CSRF), etc.
  • Scalabilité : Django est conçu pour gérer un trafic élevé et peut être facilement mis à l'échelle.
  • Versatilité : Il peut être utilisé pour créer presque n'importe quel type de site web, des systèmes de gestion de contenu aux réseaux sociaux en passant par les plateformes scientifiques.

Historique de Django

Django a été créé en 2003 par Adrian Holovaty et Simon Willison, alors qu'ils travaillaient pour le journal Lawrence Journal-World dans le Kansas. À l'époque, ils développaient des applications web pour le site d'information du journal et avaient besoin d'un framework qui leur permettrait de créer rapidement des sites web d'actualités.

Le framework a été nommé d'après le guitariste de jazz Django Reinhardt.

En 2005, il a été publié sous licence BSD et rendu open source. La première version officielle, Django 0.90, a été publiée en juillet 2005.

En 2008, la Django Software Foundation (DSF) a été créée pour maintenir et promouvoir le framework. Depuis lors, Django a continué à évoluer avec des mises à jour régulières, intégrant de nouvelles fonctionnalités et améliorations.

Versions majeures et évolution

  • Django 1.0 (septembre 2008) : Première version stable
  • Django 1.4 (2012) : Support de timezone et nouvelle application d'administration
  • Django 1.5 (2013) : Support de modèles d'utilisateurs personnalisés
  • Django 1.8 (2015) : LTS (Long Term Support) avec de nombreuses améliorations
  • Django 2.0 (2017) : Support exclusif de Python 3
  • Django 3.0 (2019) : Support de ASGI et améliorations de l'asynchrone
  • Django 4.0 (2021) : Amélioration du support de Redis, nouveau décorateur, support zoneinfo
  • Django 5.0 (2023) : Amélioration de l'asynchrone et fonctionnalités avancées

Philosophie de Django

La philosophie de Django repose sur plusieurs principes clés :

  • DRY (Don't Repeat Yourself) : Éliminer la duplication en définissant les modèles de données une seule fois.
  • Découplage élevé : Séparer les différentes parties d'une application pour améliorer la maintenance.
  • Moins de code : Maximiser la productivité en minimisant la quantité de code à écrire.
  • Explicite plutôt qu'implicite : Rendre le code clair et lisible.

Aujourd'hui, Django est l'un des frameworks web les plus populaires et est utilisé par de nombreuses entreprises de renom comme Instagram, Pinterest, Mozilla, The Washington Post, et bien d'autres.
